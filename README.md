# **Website 2 APK Builder Pro — версия 5.0**
Преобразуйте любой веб-сайт или HTML5-приложение в собственное приложение для Android.

Website 2 APK Builder Pro позволяет мгновенно превратить любой веб-сайт, HTML5-проект или локальную HTML-папку в полнофункциональное приложение для Android.  
Поддерживает ** HTML, HTML5, PHP, HTM, JS, CSS **.

## **✨ Основные моменты**

### ** Основные функции**
- Преобразование веб-сайтов в реальном времени или автономных HTML-папок в приложения для Android.
- Создание приложений, готовых для Google Play, за считанные секунды.
- Полностью ** Без рекламы ** и ** Без обратных ссылок **.
- Удобный интерфейс с быстрым процессом сборки.

### ** Улучшения пользовательского интерфейса / UX**
- Дизайн материалов с настраиваемой цветовой гаммой.
- Возможность обновления по нажатию кнопки.
- Дизайнер навигационных панелей.
- Заголовок панели инструментов в режиме реального времени.
- ** Управление значками в строке состояния (новое в версии 6.0) **.
- Пользовательская заставка и колесико прогресса.
- Управление ориентацией приложения.
- Режим рабочего стола и режим обзора.
- Поддержка масштабирования жестами.
- Улучшена обработка макета экрана при вводе текста.

### ** Функции WebView**
- ** Опция блокировки всплывающих окон (новая в версии 6.0) **.
- Улучшена обработка ошибок SSL.
- Режим работы только по протоколу HTTPS для повышения безопасности.
- Постоянные файлы cookie (поддерживает автономный режим).
- Скрытие пользовательского агента WebView (исправляет логин Google).
- Плавная работа для сайтов с большим количеством JavaScript.

### ** Медиа, загрузка и скачивание файлов**
- Загрузка файлов всех типов.
- Камера и загрузка нескольких изображений.
- Загрузка больших двоичных файлов.
- Запись звука.
- Загрузка видео с видеокамеры.
- Поддержка автономного видео HTML5.

### ** JavaScript API**
- Не допускать перехода устройства в спящий режим.
- Показывать / скрывать баннер AdMob.
- Отображение всплывающих окон и диалоговых окон.
- Действия в приложении.
- API вибрации.
- API Push-уведомлений.
- API ”isBroadcastEnabled".
- Разрешить / запретить скриншот.

### ** Уведомления**
- Push-уведомления (автоматическое расширение для длинных сообщений).
- Выберите дублирование или новый режим уведомления.

### ** Дополнительные функции**
- Поддержка пакета приложений для Android (.aab).
- Создание ** более 100 МБ APK-файлов **.
- Пользовательское хранилище ключей.
- AdMob (соответствует требованиям GDPR).
- Сохранение / загрузка файлов проекта `.webapp`.
- Поддержка макета Wix.
- Внешние ссылки открываются снаружи (необязательно).
- Подтверждение при выходе (необязательно).
- Обновлены библиотеки Firebase и AdMob.
